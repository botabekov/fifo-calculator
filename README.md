# FIFO Calculator

Небольшой одностраничный калькулятор FIFO, который работает полностью в браузере. Вся логика, UI и экспорт находятся в одном файле `index.html`.

## Быстрый старт (локально)

1. Откройте файл напрямую в браузере:
   - Дважды кликните `index.html`.

Или запустите локальный HTTP-сервер (если браузер блокирует некоторые функции при открытии файла напрямую):

```bash
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Входные данные

Ввод выполняется вручную в интерфейсе:
- Тип операции (покупка, зачисление, продажа, списание).
- Количество.
- Цена (для операций прихода и продажи; для списания цена не требуется).

Импорт Excel сейчас отсутствует — данные вводятся вручную.

## Проверка на тестовых данных

1. Добавьте пару приходов:
   - Покупка, 100 шт., 10.00
   - Покупка, 50 шт., 12.00
2. Добавьте продажу:
   - Продажа, 120 шт., 15.00
3. Нажмите «Показать расчёты» — внизу появится детализация списания по FIFO и остатки.

## Где менять формулы/логику

Основная логика FIFO и расчётов находится в `<script>` внутри `index.html`:
- Входной список операций и очереди лотов в объекте `state`.
- Обработка FIFO в функции `compute()`.
- Проверка доступного остатка в `getAvailable()`.
- Расчёт остатков в `recalcTotals()`.

Если вы хотите расширять функциональность (например, добавить импорт/экспорт), стоит вынести код в отдельные модули.
